    关于搜题软件的想法

    Created At : 2023-02-02 09:45
    Count:2.9k Views 👀 :

    我离不开搜题软件，我的同学大多数也离不开它。这种强烈的需求是有原因的，其大致可以归为两类：教辅没答案或者有答案但过分简略；学生不想通过独立思考做作业。

    教辅没答案或者有答案但过分简略的情况是很常见的。我这里所说的“教辅”，包括学生自己购买的市面上的一些题，也涵盖老师平常发的一些练习题。据我观察，在答案不全面的前提下，除抄作业外，学生使用搜题软件主要有这两个原因：一是学生做完一套题之后，急切地想知道自己正确率如何，还想把自己没有做出来的题、不知道怎么做出来的题搞懂，这时就会想尽一切办法寻找这些题的答案与解析；二是有些作业需要交上去后老师会检查，因此总有学生会担心错的太多，交上去不好看，就会想尽一切办法找答案核对一下。这是没答案的情况下使用搜题软件的原因，一种是自我反思，一种是图个安心。根据我个人体验，这两个原因都有的情况很多。

    在很多人长期使用搜题软件辅助学习的同时，也有很多人长期使用搜题软件抄答案。两款主流的搜题软件作业帮和小猿搜题都有拍整页的功能，拍摄一次，就能直接出一面的答案与解析。一些同学甚至还能对着解析在题干上圈点几笔，装作是认真思考过了。并且这样做出的作业，大多数时候老师还看不出来。学生如果长期这样做下去，最后出来的成绩不会好看。

    经过我上面的叙述，可以发现搜题软件分别有下述优缺点：

    优点

    1. 辅助真正对学习感兴趣的学生自主探索；
    2. 补足部分学生在学校受教育的短板；
    3. 能够获得海量的学生作业数据；
    4. （自身优点）与传统教辅相比数据库庞大，覆盖面广，并且有一定的用户体量，能够实时更新答案。

    缺点

    1. 助力没有兴趣学习的学生在歪路上越走越远；
    2. 难以盈利，不扩展产业、插入广告难以持续运营；
    3. 不能辅助教师教学（提供答案解析除外）。

    我向来坚持堵不如疏，因此在我看来，与其把搜题软件和学校教育划清界限，倒不如让搜题软件融入到学校教育中去，辅助学生多元发展，帮助老师了解学生情况。下面请看我不完整的方案，我会一边列举一边说明：

    1. 统一为所有学生发配“搜题设备”。由相关机构负责发配，学校要定期收回设备检查，若发现功能性障碍则要及时更换。这样的搜题设备不需要很复杂，只需能够流畅运行所有有关搜题的功能（不包括视频讲解）并且续航时间长即可，这样能把成本控制得很低（可以参考一些二手平台上四五年前的国产手机，它们中的大多数都还能流畅运行旧版软件）；
    2. 统一为所有的“搜题设备”开发一套系统和对应的软件（我的用语可能不规范）。这样的做法出于两个考虑：一是方便校方和老师在后台能查看所有学生搜题的数据；二是防止某些学生别有用心，安装其他软件，影响学习。
    3. 把学生查阅过的题上传并进行智能分析。到了这一步已经可以辅助教师教学了。教师通过研究数据，可以得知学生哪些题搜得多，进而可以根据上传图片中的书写情况了解到，哪些题学生没有完全理解，哪些题学生看不懂、做不出来（当然，长时间的线上阅卷对于任何人都非常难受，可以借助人工智能先分类一下），接下来就可以具体安排和调整教学任务了。
    4. 学生把每日做过的作业都拍照上传。上传了并不是让老师一份份看，而是通过文字识别技术进行批改分析。虽然这样一开始会因为缺乏数据样本，导致错误率很高，但是人工智能可以在日复一日的学习中不断优化，效果会越来越好。批改分析之后，学生能拿到带有错题解析的结果，理解不透彻的题还可以再进一步看解析，并且设备还能给出错题的同类型题让学生巩固练习；老师在大幅度省略批改这一步骤后，仍然能知道学生整体的作业完成情况，并针对此调整教学。
    5. 软件针对错误率最高的题定期出卷子让学生巩固练习。这点需要每个学生都进行实名认证，将账号和自己的学籍绑定，软件定期针对学生一段时间做错的题出卷子，并让学生自愿选择是否巩固练习。

    以上是我对这种构想非常简略也非常不全面的一个叙述，乍一看或许非常美好，但是推行中会遇到许多问题，下面让我来总结一下这样做的优缺点，先给自己泼一盆冷水。

    优点

    1. 协助老师教学，减轻老师工作压力，促进教学科学化；
    2. 助力有能力学生自学；
    3. 能真正将搜题软件和辅助学生学习结合起来。

    缺点

    1. “搜题设备”的制造、发配、运营、维护、回收需要巨大的产业体系支撑。“搜题设备”的软件对硬件要求不高，但像这样的电子设备也最多能陪伴一个学生从初一上到高三，也就是六到七年的时间，到时候就需要淘汰掉。设备发到学生手中后，就是他们自己拿着了，丢了、损坏等事件会屡见不鲜。因此这是一个巨大的难点；
    2. 个人隐私的保护。如果新开发一套软件架构使用，并且和每个学生的学籍绑定，虽然便利了题目信息的收集，但也带来了巨大的个人隐私泄露风险；
    3. 如果有学校或地市用搜集到的数据来出试卷考试，不排除有别有用心的人大量上传低级错题，污染数据库；
    4. 这项技术只能帮助大多数对于学习有主动性的同学，对于厌学的同学起不到激励学习的作用。

    虽然实现的过程中会遇到种种问题，但如果真的有解决种种问题实现了这个构想的一天，会不会很美好呢？让我来带着大家想象一下：

    中学生小明结束了他一天的课程，接下来是自习时间。对于学习成绩不错的他来说，在晚自习的时间完成晚上的作业绰绰有余。他决定先写理科作业。他在写一道数学几何题时卡壳了，想了20分钟也没想出来。他跳过这道题。他做完后面的题之后再回来想，仍然没有想出来。他拿出“搜题设备”拍照，设备自动识别出了这道题没有写，在题库里找到了这道题的解析后就推给了小明。小明看完解析之后恍然大悟，还把这道题加入了“电子错题本”。小明写完所有作业后，按照科目把作业在软件中拍照上传，紧接着开始做一些其他教辅。小明拍摄的图片和对应题目的详细信息，会和其他同学拍摄的图片一块上传到数据库中，通过分析每天的数据，就能知道哪些题易错，哪些题学生不理解，从而帮助老师教学；还能根据这些信息建立学生档案，对每个学生的具体情况有针对性的安排训练。小明给自己安排的任务完成之后，使用设备把自己刚刚做过的教辅一张张拍照上传，这样既能核对答案，又能丰富他个人的学生画像，使得设备更有针对性的安排练习，还能更广泛的搜集优秀的题供所有人练习。有的同学直接不写作业，选择拍整页抄答案，这些空白照片在上传后都会自动统计。老师拿到这些学生的名单后，可以单独询问原因，及时纠正学生错误的学习态度，如果有其他原因的话，也能很快化解误会。如果不上传照片，照样可以统计。到了节假日，小明想巩固一下自己学的不扎实的知识，设备根据他平常上传照片形成的的用户画像，给他出了七科卷子，外加一套私人定制的体育训练计划。小明非常快乐，他相信设备是辅助他学习的好手，自己上传的题越多，设备对他的用户画像就越详细，给他出的题也就更有针对性，自己也就越能在这个过程中提升自己，发觉学习的乐趣。

    想象完了，的确很美好，但这是最理想化的状态，是建立在“所有学生都相信在‘搜题设备’中搜题，能丰富自己的用户画像，从而使得设备给自己出的题更有针对性，对自己的成绩更有帮助”的前提下的。有的学生拒绝使用“搜题设备”怎么办？有的学生根本不学习怎么办？有的学生不经过搜题设备查询答案，直接抄其他同学的怎么办？有的学生内卷到极致，不停地让“搜题设备”给自己出题，学坏了身体怎么办？有的学生不信任“搜题设备”怎么办？有的学生故意破坏自己和他人的“搜题设备”怎么办？学生的个人账号被盗用怎么办？老教师掌握不了用后台管理所有“搜题设备”怎么办？有的教师过度依赖“搜题设备”只让学生通过做题来提高成绩，怎么办？老师或者学生因为生理因素无法使用搜题设备怎么办？校方落实人手一台“搜题设备”不到位怎么办？不良媒体煽动群众情绪，宣扬“阴谋论”，要平息舆论，解决人民群众担忧怎么办？“搜题设备”落实后，各地的教育差距拉不开了，有的人会怎么办……有太多的问题需要思考。

    但这确实是我眼中可预见的一种比较光明的未来。我希望，未来的教学，除了能够让所有的学生认为学校里和同龄人一起成长是快乐的，还要让他们认识到一起学习也是快乐的，通过学习提升自我是最快乐的。

    Chaoran
